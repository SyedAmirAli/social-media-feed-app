1. প্রদত্ত HTML/CSS ডিজাইন **React.js বা Next.js**-এ কনভার্ট করতে হবে। ডিজাইন পরিবর্তন করা যাবে না।

2. একটি **সিকিউর Authentication System** তৈরি করতে হবে (JWT বা Session-based)।

3. **Registration**-এ থাকতে হবে:
    - First Name
    - Last Name
    - Email
    - Password

4. **Login** সিস্টেম তৈরি করতে হবে।

5. শুধুমাত্র লগইন করা ইউজার **Feed Page** অ্যাক্সেস করতে পারবে (Protected Route)।

6. Feed-এ সকল ইউজারের **Public Post** এবং নিজের **Private Post** দেখতে পারবে।

7. পোস্টগুলো **Newest First** (সর্বশেষ পোস্ট আগে) দেখাতে হবে।

8. নতুন **Text + Image Post** তৈরি করা যাবে।

9. প্রতিটি পোস্টে **Public / Private Visibility** সেট করা যাবে।
    - Public → সবাই দেখতে পারবে।
    - Private → শুধুমাত্র পোস্টের মালিক দেখতে পারবে।

10. **Post Like / Unlike** সিস্টেম থাকতে হবে।

11. **Comment** যোগ করা যাবে।

12. প্রতিটি **Comment-এ Reply** করা যাবে।

13. **Comment Like / Unlike** সিস্টেম থাকতে হবে।

14. **Reply Like / Unlike** সিস্টেম থাকতে হবে।

15. Post, Comment এবং Reply-তে **কারা Like করেছে** তা দেখানোর ব্যবস্থা থাকতে হবে।

16. Backend যেকোনো প্রযুক্তি দিয়ে করা যাবে।

17. Database যেকোনো ব্যবহার করা যাবে।

18. Security Best Practices অনুসরণ করতে হবে।

- Password Hashing
- Authorization
- Input Validation
- Protected Routes

19. Performance মাথায় রেখে ডেভেলপ করতে হবে, যেন ভবিষ্যতে **মিলিয়ন পোস্ট ও রিড** হ্যান্ডেল করা সম্ভব হয় (যেমন: ভালো Database Design, Indexing, Pagination ইত্যাদি)।

20. Code-এর জন্য **GitHub Repository** দিতে হবে।

21. অ্যাপের **YouTube Video Walkthrough** (Unlisted/Private) দিতে হবে, যেখানে সব ফিচার দেখাতে হবে।

22. সম্ভব হলে অ্যাপ **Live Deploy** করে URL দিতে হবে (Recommended)।

23. একটি সংক্ষিপ্ত **Documentation** দিতে হবে, যেখানে থাকবে:

- কী কী তৈরি করা হয়েছে
- কী Tech Stack ব্যবহার করা হয়েছে
- কী কী Design/Architecture Decision নেওয়া হয়েছে
- Project কীভাবে Run করতে হবে
